  1. 创建一个组，要求组id为1200，组名称为devteam。展示创建的组信息。修改组的名称和id并展示。删除刚才创建的组。 
bash
# 创建组
sudo groupadd -g 1200 devteam

# 展示创建的组信息
getent group devteam

# 修改组的名称
sudo groupmod -n newdevteam devteam

# 修改组的ID
sudo groupmod -g 1300 newdevteam

# 展示修改后的组信息
getent group newdevteam

# 删除组
sudo groupdel newdevteam
  1. 创建一个用户，要求用户id为默认，路径修改为/home/XXX，附加组为root组。展示刚才创建的内容，修改主组为root组并展示。删除刚才创建的用户。 
bash
# 创建用户（默认ID）
sudo useradd -m -d /home/XXX -G root newuser

# 展示创建的用户信息
id newuser

# 修改主组为root组
sudo usermod -g root newuser

# 展示修改后的用户信息
id newuser

# 删除用户
sudo userdel newuser
  1. 创建一个用户和组，将刚刚创建的用户添加到刚刚创建的组中（作为附加组），展示组里的成员，删除刚才添加进的用户。 
bash
# 创建组
sudo groupadd mygroup

# 创建用户并添加到组
sudo useradd -G mygroup myuser

# 展示组里的成员
getent group mygroup

# 删除用户
sudo userdel myuser

# 删除组
sudo groupdel mygroup
  1. 创建一个用户，通过vim将root用户的密码赋予刚才创建的用户，显示一下刚刚创建的用户密码信息，最后锁住用户的密码。 
bash
# 创建用户
sudo useradd myuser

# 设置用户密码（通过vim）
sudo passwd myuser

# 显示用户密码信息
sudo cat /etc/shadow | grep myuser

# 锁住用户密码
sudo passwd -l myuser

bash
# 创建一个文件
touch my_file.txt

# 修改文件的用户和组
